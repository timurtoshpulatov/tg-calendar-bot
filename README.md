# TG Calendar Bot 🗓

Telegram-бот-календарь с ИИ-парсером задач и повторяющимися задачами.

## Возможности

- 🧠 **ИИ-парсинг** — понимает обычный язык: «встреча с другом завтра в 15:00»
- 🔁 **Повторяющиеся задачи** — «тренировка каждый понедельник в 18:00»
- ✅ **Выполнение по названию** — «выполнил задачу встреча с другом»
- ⏰ **Напоминания** — за 60 минут до дедлайна
- 📋 **Утренняя сводка** — в 09:00

## Быстрый старт (локально)

```bash
pip install -r requirements.txt
python bot.py
```

## Деплой на Railway (24/7)

### 1. Создай репозиторий на GitHub
- Загрузи все файлы проекта

### 2. Зарегистрируйся на Railway
- Перейди на [railway.app](https://railway.app)
- Авторизуйся через GitHub

### 3. Создай проект
- Нажми **New Project** → **Deploy from GitHub repo**
- Выбери свой репозиторий

### 4. Настрой переменные окружения
В настройках проекта (Variables) добавь:

| Переменная | Значение |
|------------|----------|
| `BOT_TOKEN` | Твой токен бота от @BotFather |
| `AI_API_KEY` | Твой ключ OpenRouter |

### 5. Деплой
- Railway автоматически задеплоит бота
- Бот будет работать 24/7
- Автоматически обновляется при пуше в GitHub

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен Telegram бота | - |
| `AI_API_KEY` | API ключ OpenRouter | - |
| `AI_MODEL` | Модель ИИ | `nvidia/nemotron-3-ultra-8b` |
| `DIGEST_TIME` | Время утренней сводки | `09:00` |
| `REMINDER_MINUTES` | За сколько минут напоминать | `60` |
